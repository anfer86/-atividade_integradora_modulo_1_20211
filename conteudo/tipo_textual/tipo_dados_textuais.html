IFSC - PROJETO INTEGRADOR DE BANCO E DADOS

CHAR

CHAR é um tipo de dado textual utilizado em SGBD's para determinar o conteúdo de uma coluna com comprimento fixo. Usamos o CHAR para armazenar dados em uma coluna com um certo número de caracteres sem ultrapassar o valor estabelecido. Se armazenarmos apenas um caractere em uma coluna CHAR(10) por exemplo, os outros nove espaços restantes serão preenchidos com espaços em branco.

EXEMPLO

VARCHAR

VARCHAR é um tipo de dado textual utilizado em SGBD's para determinar o conteúdo de uma coluna com o comprimento variável. Usamos o VARCHAR para armazenar dados com tamanhos variáveis. Se armazenarmos apenas um caractere em uma coluna VARCHAR(10) por exemplo, ele vai armazenar somente o caractere sem acrescentar espaços em branco para completar o tamanho definido.

EXEMPLO

TEXT

TEXT é um tipo de dado textual utilizado em SGBD's para determinar o conteúdo de uma coluna com o comprimento sem limite específico de caracteres. Usamos TEXT para armazenar grandes quantidades de caracteres.

EXEMPLO

Exemplo dos Códigos:

create table diretor (

id_diretor int primary key,

create table estudio (
