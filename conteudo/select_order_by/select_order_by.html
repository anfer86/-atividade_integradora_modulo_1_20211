ORDER BY em SQL

A tradução em português mais próxima do inglês do comando "Order By" significa "Ordenar por".

ORDER BY organiza os resultados de acordo com uma ou mais colunas de uma tabela, podendo definir a ordem dos resultados, utilizando:

  • ASC : Definindo a ordem como Crescente.

  • DESC : Definindo a ordem como Descrente.

    • Acompanhe o exemplo abaixo!

    Como exemplo vamos utilizar esta tabela de uma pizzaria chamada de tabela "pizza":

    id_pizza nome preco calorias glutem id_tipo(FK) dt_hr_cadastro
    1 Margherita 27 1500 TRUE 1 2021-08-27 17:00:00
    2 Calabresa 29 null TRUE 1 2021-08-28 17:05:00
    3 Pepperoni 31 1900 TRUE 1 2021-08-29 17:10:00
    4 Camarão 49 1600 TRUE 2 2021-08-30 17:15:00
    5 Filé com Cheddar 50 2400 TRUE 2 2021-08-31 17:20:00
    6 Tomate Seco 44 1500 FALSE 2 2021-09-01 17:25:00
    7 Chocolate 27 1900 TRUE 3 2021-09-03 17:30:00
    8 Paçoca 29 2300 TRUE 3 2021-09-03 17:35:00

    Note que a coluna dos Preços(preco) está "bagunçada", veja a diferença nos seguintes comandos e tabelas:

    SELECT nome, preco FROM pizza;
    nome preco
    Margherita 27
    Calabresa 29
    Pepperoni 31
    Camarão 49
    Filé com Cheddar 50
    Tomate Seco 44
    Chocolate 27
    Paçoca 29
    SELECT nome, preco FROM pizza
    ORDER BY preco DESC;
    nome preco
    Filé com Cheddar 50
    Camarão 49
    Tomate Seco 44
    Pepperoni 31
    Calabresa 29
    Paçoca 29
    Margherita 27
    Chocolate 27

    Na segunda tabela criamos uma tabela utilizando o ORDER BY nos preços das pizzas, e organizamos do maior preço para o menor com o DESC (poderia usar ASC para obter o resultado oposto da consulta).
