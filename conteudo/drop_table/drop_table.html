Comando Drop Table SQL

Drop table é um dos códigos mais simples de SQL ele apenas tem uma única utilidade, que é deletar uma tabela

EXEMPLO 1

você digitou "igreniente" ao invés de "ingrediente" em uma tabela e para corrigir o erro não pode criar outra tabela, pois ela já existe sendo assim você deve usar o comando DROP TABLE para destruir a tabela e criar uma nova

basta dar um drop table igreniente. que a tabela será deletada

EXEMPLO 2

a tabela "id_tipo" não se encaixa no banco de dados e você quer remove-la, sendo assim, basta dar um DROP TABLE tipo. para deletar ela o banco de dados

pode se usar também um DROP TABLE IF EXISTS tipo. o resultado é o mesmo, a diferença é que se a tabela não existir não retorna erro.
