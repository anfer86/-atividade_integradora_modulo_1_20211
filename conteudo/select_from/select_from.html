SQL select

A linguagem SQL foi criada com o objetivo de padronizar os comandos de manipulação de dados em SGBD´s. Conhecendo bem a linguagem, é possivel acessar os recursos básicos de qualquer banco de dados relacional como oracle, SQL server ou MYSQL, sem praticamente nenhuma mudança. Nesse site veremos como utilizar o SELECT que é muito importante para as consultas SQL.

Select "Iniciante"

O comando SELECT tem a função de retornar um conjunto de resultados de registros de uma ou mais tabelas, ele recupera zero linhas ou várias linhas de uma ou mais tabelas. O comando mais básico é: select (colunas) from (nome da tabela)

Exemplo 1

Select * From tipo.

No caso do banco de dados utilizado para fazer esse site o resultado seria:

id_tipo tipo
1 Tradicional
2 Especial
3 Doce

O caractere * utilizado no select significa que eu estava selecionando todas as linhas e todas as colunas da tabela.
