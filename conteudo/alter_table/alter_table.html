Instrução: Alter Table!

A Instrução Alter Table é Usada Para:

- Adicionar

- Excluir

- Modificar

- Adicionar Restrições

- Remover Restrições

Assim, modificando uma tabela existente!

Codigos MySQL!

Exemplos de Comandos que usam Alter Table:

ADD column_name datatype;

- Podemos adicionar uma nova Coluna na tabela Pizza com o seguinte comando:

ALTER TABLE Pizza ADD Promoção BOOLEAN;

RENAME COLUMN old_name TO new_name;

- Podemos Alterar o nome de uma Coluna com o seguinte comando:

ALTER TABLE Pizza RENAME COLUMN Nome TO Pizzas;
