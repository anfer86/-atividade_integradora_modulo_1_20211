Comando insert Linguagem SQL

O comando Insert, quando usado possui um "estilo base", seria este: INSERT INTO (tabela) VALUES (valores para cada coluna separados por vírgula).

O insert é um comando que insere os dados determinados nas colunas respectivas.

Um exemplo de insert na tabela a seguir:

pizza

id_pizza nome preco calorias glutem id_tipo (FK) dt_hora_cadastro
1 Margherita 27 1500 TRUE 1 2021-08-27 17:00:00
2 Calabresa 29 null TRUE 1 2021-08-28 17:05:00
3 Pepperoni 31 1900 TRUE 1 2021-08-29 17:10:00
4 Camarão 49 1600 TRUE 2 2021-08-30 17:15:00
5 Filé com Cheddar 50 2400 TRUE 2 2021-08-31 17:20:00
6 Tomate Seco 44 1500 FAlSE 2 2021-09-01 17:25:00
7 Chocolate 27 1900 TRUE 3 2021-09-02 17:30:00
8 Paçoca 29 2300 TRUE 3 2021-09-03 17:35:00

É: INSERT INTO pizza VALUES (9, "Jujuba", 25, 1600, "Falso", 3, "2021-08-03 23:00");

  • O [9] é inserido em id_pizza.
  • O ["Jujuba"] inserido em nome.
  • O [25] inserido em preco.
  • O [1600] inserido em calorias.
  • O ["FALSE"] inserido em glutem.
  • O [3] inserido em id_tipo, que é uma chave estrangeira mencionada com (FK), da tabela seguir:

    tipo

    id_tipo tipo
    1 Tradicional
    2 Especial
    3 Doce
  • O ["2021-09-04 17:40:00"] inserido em dt_hr_cadastro.
